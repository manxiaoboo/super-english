# go to work

- 发音：/ɡoʊ tə wɜːrk/
- 词性：verb phrase
- 核心意思：to travel to your workplace; to start your working day
- 核心动作/画面：整个去上班的过程——从出门到到达工位
- 中文释义：去上班；上班

## 荒诞寓言

一只企鹅每天早上都要从冰屋步行到海边的办公楼。它的路线是：出门→滑过冰面→跳三块浮冰→在企鹅咖啡店买杯鱼露拿铁→刷卡进办公室。有一天它出门晚了，海豹问它：你 leave for work 了吗？企鹅说：刚刚出门！十分钟后海豹在办公室看到它，说：你到得真快。企鹅说：是啊，我从 leave for work 到 go to work 的整个过程只用了十分钟。

## 寓意

leave for work = 出发（时间点）；go to work = 去上班（整个过程，包含出发+通勤+到达）。

## 记忆钩子

leave for work = leave（离开）+ for（前往）→ 只说出出发。go to work = go（去）+ to（到）→ 包含从出门到到达的全部。

## 易混词/注意点

go to work vs leave for work（对比）：leave for work 强调出发的时间点；go to work 强调整个上班过程（包含出发+通勤+到达）。
go to work vs get to work：get to work 强调到达（arrive at work）。
注意：go to work 是固定短语，不加 the。
