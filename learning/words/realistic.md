# realistic

- 发音：/riːəlɪstɪk/
- 词性：adjective
- 核心意思：based on what is actually possible or true, not on what you wish were true
- 核心动作/画面：把气球里的空气放掉一半，脚踩在地上——不飘，但也没倒
- 中文释义：现实的；实际的；切实可行的；（也指）逼真的、写实的

## 荒诞寓言

有只蜗牛报名参加了百米短跑。它每天训练四小时，充满热情。比赛前，教练把它叫到一边，轻声说：我们来聊聊 realistic 的目标。蜗牛愣住了。教练说：你的爆发力提升了百分之三百，但你依然是一只蜗牛。蜗牛沉默了三秒，然后改报了蜗牛组比赛。结果它以绝对优势拿了金牌，还破了世界纪录。赛后它说：选一个 realistic 的目标，比拼命追一个错误的目标更聪明。

## 寓意

realistic 不是放弃梦想，而是对自己的处境和能力有清醒的认知，然后在这个基础上全力以赴。

## 记忆钩子

real（真实的）+ -istic（形容词后缀，表具有……性质的）→ 具有真实性的 → realistic = 基于现实的、实际的（真实词根构成）

## 英文锚定句

We need to set realistic deadlines instead of promising delivery dates we cannot meet.

## 常见搭配

- realistic expectations / goals / targets / deadlines
- a realistic assessment / estimate / approach / plan
- be realistic about something
- realistic graphics / simulation / portrayal
- let us be realistic (spoken)

## 个人化例句

Before committing to a sprint scope, I always try to set realistic estimates based on past velocity, not best-case assumptions.

## 易混词/注意点

realistic vs practical: practical focuses on operability; realistic focuses on whether something is achievable given current conditions. realistic vs pessimistic: realistic is accurate assessment; pessimistic is underestimating possibility. Adverb: realistically — Realistically, we cannot finish this in one week.
