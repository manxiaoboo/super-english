## Meaning

give more attention or importance to something

## Usage Context

- 中文：更在意；更看重
- 常见结构：care more about + noun / noun phrase
- 很适合表达价值取向：care more about individuality / quality / value for money
- 在这次翻译里，它比 pursue the personality 更自然

## Anchor Sentence

Some buyers care more about individuality than famous brands.
