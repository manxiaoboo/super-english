## Meaning

the actual link, not a shortened, hidden, or displayed label

## Usage Context

- 中文：真实链接；实际链接
- 常见场景：说明文档、网页源码、Markdown 文档、重定向链接
- 常见搭配：see the real URL, copy the real URL, check the real URL in the source
- 和 actual URL 很接近；real URL 更口语、更直接

## Anchor Sentence

You can see the real URL in the original Markdown document.
