## Meaning

immediately; at this moment

## Usage Context

- 中文：现在；立刻；马上
- 核心强调：强调时间，就是现在这个时刻
- 常带一点催促感、紧迫感，尤其在口语里
- 常见用法：I need it right now. / We have to leave right now. / I can't talk right now.

## Anchor Sentence

We need to fix this right now.
