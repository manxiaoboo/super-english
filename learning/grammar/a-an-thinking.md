## 核心思维

a/an 的核心不是数量上的「一个」，而是**从类别中挑出一个任意代表**。说话者脑子里先有一个类别概念（如 dog），然后从中挑出任何一个个体来谈论——这个个体对听话者是**新信息**（第一次出现）。

三步思维路径：
1. 脑子里有一个类别概念（"dog"）
2. 从类别中挑出一个代表（any one）
3. 这个代表对听话者是**新信息**

## 结构图解

```
a/an 的思维路径：

类别概念 ──挑出任意一个──→ 新信息
  "dog"        a dog      "你之前不知道是哪条狗"

四种主要用法：
┌──────────────────────────────────────────────────┐
│ 1. 首次提到  → I bought a car yesterday.         │
│ 2. 泛指类别  → A cat has sharp claws.            │
│ 3. 表示"某个"→ A man is waiting for you outside. │
│ 4. 价格/频率 → twice a week, $5 a kilo          │
└──────────────────────────────────────────────────┘

不加 a/an 的情况：
┌──────────────────────────────────────────────────┐
│ × 不可数名词前面   → I need water.               │
│ × 复数名词前面     → I like dogs.                │
└──────────────────────────────────────────────────┘

注意 an 规则：元音音素开头用 an（不只是元音字母）
  - an honest man (h 不发音, /ɒnɪst/)
  - a university (以 /j/ 开头, 辅音)
```

## 锚定例句

1. **I saw a bird in the tree.**
   ```
   a bird → 首次提到，听话者不知道是哪只鸟
   ```
   如果继续说 The bird was singing，第二次提到就用 the。

2. **A good teacher always listens to students.**
   ```
   a good teacher → 泛指类别 = 任何一个好老师
   ```
   不是说某个特定的老师。

3. **She wants to be an engineer.**
   ```
   an engineer → 职业 = 属于工程师这个类别的一员
   ```
   职业名词前必须用 a/an。

## 细微差别

🔍 **a doctor（任何一个医生）vs. the doctor（那个医生）**

- **I need a doctor.** → 我需要一个医生，可以是任何一个
- **I need the doctor.** → 我需要那个特定的医生（比如你的家庭医生）

🔍 **an + 元音音素 vs. a + 辅音音素**

- **an honest man** → honest 发音 /ˈɒnɪst/，h 不发音，以元音开头
- **a university** → university 发音 /ˌjuːnɪˈvɜːrsəti/，以 /j/ 辅音开头
