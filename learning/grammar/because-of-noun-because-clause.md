# because of + noun / because + clause

## Meaning

Use because of before a noun phrase, and use because before a full clause.

## Usage Context

- because of + noun / noun phrase
- because + subject + verb
- 这次翻译里：because they trust them 是完整从句；because of trust 也可以，但后面不能直接接完整句子

## Anchor Sentence

People buy famous brands because they trust them.

## Review Questions

1. because of 后面接名词短语，还是完整句子？
2. 把 because of trust 和 because they trust them 的差别说出来。
3. 用 because 或 because of 各写一句原因句。
