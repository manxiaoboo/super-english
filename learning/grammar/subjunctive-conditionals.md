## 核心思维

英语用"时态后退"来标记"这件事不是真的"。真实条件句用正常时态（If + 一般现在时, will + 原形），虚拟条件句用时态向过去退一格的方式（If + 一般过去时, would + 原形）来暗示这是反事实假设。所有人在虚拟语气中，be 动词统一用 were（包括 I/he/she/it）。

## 结构图解

```
         真实 → 正常时态
if 从句 ──┤
         虚拟 → 时态后退一格（现在→过去，过去→过去完成）

         真实 → will / can / may
主句   ──┤
         虚拟 → would / could / might
```

## 锚定例句

1. If I **were** rich, I **would travel** the world.  
   （如果我有钱，就环游世界。——事实：我没钱）

2. If she **lived** closer, we **would see** each other more often.  
   （如果她住得近一点，我们就能经常见面。——事实：她住得远）

3. If I **were** you, I **would not quit** the job right now.  
   （如果我是你，我现在不会辞职。——事实：我不是你）

## 最小对比

| | 真实条件 | 虚拟条件 |
|---|---|---|
| 例句 | If I **have** time, I **will** come. | If I **had** time, I **would** come. |
| 含义 | 可能有时间，在做计划 | 没时间，表达遗憾/愿望 |
| 时态 | 一般现在时 → will | 一般过去时 → would |

## 复习问题

1. **辨识**：If it rains tomorrow, the picnic will be cancelled. vs If it rained tomorrow, the picnic would be cancelled. ——哪句是虚拟？
2. **填空**：If I ______ (be) you, I ______ (take) that offer.
3. **造句**：中文：如果我会飞，我每天飞去上班。——用虚拟语气翻译。

## 常见陷阱

- 把虚拟当真实：明明不可能却用 If I have time（应该是 had）
- 时态混搭：If I had time, I will help you ❌（前后时态不呼应，应为 would）
- was 代替 were：If I was you ❌（正式虚拟语气中所有人称用 were）
