## Meaning

说明两个完整句子不能只用逗号硬连，需要 and、but、so、where 等连接方式

## Usage Context

- 两个完整句子不要只用逗号连在一起
- 可以用 and / but / so 连接
- 也可以改成从句，比如 where you can see ...
- 这次翻译里：Here is the original Markdown file, you can see ... 不自然；更自然的是 Here is ..., and you can see ... 或 Here is ..., where you can see ...

## Anchor Sentence

Here is the original Markdown document, and you can see the real URL in it.

## Review Questions

1. 为什么 Here is ..., you can see ... 不自然？
2. 把两个完整句子连起来时，可以用哪些方式？
3. 把 Here is the source file, you can see the full link there. 改成更自然的英文。
