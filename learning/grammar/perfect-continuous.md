## 核心思维

完成进行时 = "回望" + "持续"的杂交：站在某个时间点回望，一个动作从更早开始一直延续到该时间点，并且强调这个动作的连续性。完成式说"做完了没"，完成进行式说"一直在做"——前者关心结果，后者关心过程。

## 结构图解

```
have/has/had/will have + been + doing

现在完成进行：    过去起点              现在
                     ↓                   ↓
                     ●═══════════════════▶
                     │── 一直在等待 ────│   

过去完成进行：    had been doing   → 站过去回望持续
将来完成进行：    will have been doing → 站将来回望持续
```

## 锚定例句

1. I have been studying English for two hours.
   → 两小时前开始学，持续到现在，可能还会继续。

2. She has been working here since 2019.
   → 2019入职，至今还在，强调"从未间断"。

3. By next July, I will have been learning English for a full year.
   → 站明年7月回望，已经持续学了整整一年。

## 最小对比

```
结果 vs 过程：
A: I have read this book.（读完了——强调结果）
B: I have been reading this book.（一直在读——可能还没读完）

被动 vs 主动：
A: She has been crying.（主动：她一直在哭）
B: She has been cried. ❌（被动：被哭了——不通！）

by+将来：
A: So far, I have been working for 6 hours.（so far=站现在）
B: By 9pm, I will have been working for 12 hours.（by 9pm=站未来）
```

## 复习问题

1. 辨识：I have cleaned the kitchen vs I have been cleaning the kitchen 区别是什么？
2. 填空：By midnight, I ___ (work) for 12 hours straight.
3. 造句：用现在完成进行时写一句描述你一直在做的一件事。

## 常见陷阱

- been doing（主动进行）≠ been done（被动）：has been crying ≠ has been cried
- by + 将来时间 → will have (been) doing，不是 have (been) doing
- 状态动词（know/like/believe）不用进行式
- 宾语是时间段用 been doing，宾语是结果对象用 done
