## 核心思维

英语名词的真正分类不是「可数/不可数」，而是名词背后代表的三类**世界观**。同一个名词可以在不同世界观之间切换，可数性随之变化。与其死记一个词「可数还是不可数」，不如理解它此时代表的是哪种存在。

## 结构图解

```
英语名词的三种世界观：

  个体名词 Individual        物质名词 Mass          抽象名词 Abstract
  ──────────────────        ──────────────         ──────────────────
  "独立存在的个体"           "连续的、无边界的东西"    "看不见摸不着的概念"

  dog, chair, book          water, air, rice       love, time, knowledge
  → 天然可数                → 天然不可数             → 一般不可数

  跨界切换：
  ┌──────────────────────────────────────────────────┐
  │ water (物质) ──加容器──→ a glass of water (个体)  │
  │ coffee (物质) ──省略容器─→ a coffee (个体: 一杯咖啡)│
  │ experience (抽象) ──具体化─→ an experience (个体)  │
  │ beauty (抽象) ──指代人──→ a beauty (个体: 一个美人) │
  └──────────────────────────────────────────────────┘
```

## 锚定例句

1. **I drink coffee every day.**
   ```
   coffee → 物质名词（咖啡这种饮料，不可数）
   ```
   每天喝咖啡，说的是咖啡这种「东西」。

2. **I'd like a coffee, please.**
   ```
   a coffee → 个体名词（= a cup of coffee，可数）
   ```
   点单时说 a coffee，把咖啡「个体化」为一杯咖啡。

3. **She has experience with this kind of work.**
   ```
   experience → 抽象名词（经验这个概念，不可数）
   ```

4. **She had a great experience last year.**
   ```
   an experience → 个体名词（一次具体的经历，可数）
   ```

## 最小对比

| 物质/抽象用法（不可数） | 个体用法（可数） |
|---|---|
| I need some **paper** to write on.（纸这种材料） | I read an interesting **paper** yesterday.（一篇论文） |
| There is **hair** on the floor.（地上有头发——物质） | There is **a hair** in my soup.（一根头发——个体） |
| I don't have **time**.（时间这个概念） | I've been there three **times**.（三次） |
| **Beauty** is subjective.（美这个概念） | She is **a beauty**.（一个美人） |

核心规律：**英语允许你给「物质/抽象」加上边界或具体化，一旦具体化，就变成可数的个体。**

## 复习问题

1. 英语名词的三种世界观分别是什么？各举一个例子。
2. 为什么同一个 coffee 在 "I drink coffee" 中不可数，在 "a coffee" 中可数？
3. 填空：I don't have much _______ with this software. (experience / experiences)
4. 填空：I had many interesting _______ during my trip. (experience / experiences)
5. 用 experience 的两种世界观各造一个句子。

## 常见陷阱

中文思维中「经验」和「经历」是两个词，但英语只有一个 experience。学习者容易混淆什么时候可数、什么时候不可数。口诀：**泛指经验不可数，具体经历可数用。**
