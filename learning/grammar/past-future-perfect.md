## 核心思维

过去完成时 = "过去的回望"：站在过去某时刻，回头看更早发生的事。将来完成时 = "将来的回望"：站在将来某时刻，回头看那时已完成的事。本质是回望站的位置不同——现在完成站现在回望，过去完成站过去回望，将来完成站未来回望。

## 结构图解

```
过去完成：had + 过去分词
将来完成：will have + 过去分词

时间轴三站回望：

过去完成          现在完成           将来完成
had + done     have/has + done    will have + done
    ↓               ↓                  ↓
●══════▶ ●     ●══════▶ 现在     ●══════▶ ●
(更早) (过去)   (过去)            (将来某时刻)
```

## 锚定例句

1. When I arrived, she had already left.
   → arrived = 过去锚点，had left = 站在"到达"时刻回望，走在那之前。

2. By the end of this year, I will have learned 500 English words.
   → end of this year = 将来锚点，will have learned = 站在年底回望，词汇已积累完成。

3. I realized I had forgotten my wallet at home.
   → realized = 过去锚点，had forgotten = 在"意识到"之前已经忘了。

## 最小对比

```
过去完成 vs 一般过去：
A: When I arrived, she left.（我到的时候，她走了——可能同时）
B: When I arrived, she had left.（我到的时候，她已经走了——她先走，我后到）

将来完成 vs 一般将来：
A: I will finish the report by Friday.（周五前会完成——关注动作）
B: I will have finished the report by Friday.（到周五时报告在手里——关注结果已就绪）
```

## 复习问题

1. 辨识：By the time you read this, I will have left the city. 用了什么时态？站在哪个时间点回望？
2. 填空：By the end of this year, I ___ (learn) 500 English words.
3. 造句：用过去完成时写一句，描述"你过去某个时刻发现，在那之前已经发生了某事"。

## 常见陷阱

- 过去完成必须搭配一个过去时间锚点，不能孤立使用
- 将来完成常用 by + 将来时间（by Friday, by next month）标记锚点
- by the end of this year 用 will have done，不是 have done
- 过去完成从句和主句的时态必须一致（都在过去时间轴上）
