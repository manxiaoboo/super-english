## 核心思维

可数名词复数的本质不是「加 -s 的拼写规则」，而是**英语对个体数量显式标记的思维习惯**。既然 apple 是独立存在的个体，那有多个时，语言上必须显式标出来——中文靠数量词暗示，英语要求名词自身变形标记。

## 结构图解

```
可数名词复数体系

  ┌─ 规则复数 ─────────────────────────────────┐
  │  拼写：-s / -es / -ies                      │
  │  发音：/s/(清辅音后) /z/(浊辅音后) /ɪz/(咝音后)│
  │  例：books /s/  dogs /z/  buses /ɪz/        │
  └─────────────────────────────────────────────┘

  ┌─ 不规则复数 ────────────────────────────────┐
  │  元音变化：man→men, tooth→teeth, mouse→mice │
  │  -(r)en型：child→children, ox→oxen          │
  │  f→ves型：knife→knives, wife→wives          │
  │  外来词型：analysis→analyses, crisis→crises  │
  └─────────────────────────────────────────────┘

  ┌─ 单复数同形 ────────────────────────────────┐
  │  sheep, deer, fish, series, species, means   │
  │  形态不变，靠上下文判断单复数                   │
  └─────────────────────────────────────────────┘

  ┌─ 天生复数 ──────────────────────────────────┐
  │  trousers, scissors, glasses, clothes        │
  │  成对/多部分构成的物品，只有复数形式             │
  └─────────────────────────────────────────────┘
```

## 锚定例句

1. **The children brushed their teeth after the meal.**
   ```
   children ← child ( -(r)en 型不规则 )
   teeth    ← tooth ( 元音变化型 )
   ```
   两个不规则复数出现在同一句中。

2. **We need several sharp knives.**
   ```
   knives ← knife ( f → ves 型 )
   发音：/naɪvz/ — 注意 v 和 /z/，不是 /s/
   ```

3. **Three sheep are in the field, and one sheep is under the tree.**
   ```
   sheep → sheep（单复数同形）
   靠 are/is 区分数量。
   ```

## 最小对比

| 单数 | 复数 | 类别 |
|------|------|------|
| one child | two **children** | -(r)en 型 |
| one mouse | three **mice** | 元音变化 |
| one knife | four **knives** | f→ves |
| one analysis | several **analyses** | 外来词 |
| one sheep | five **sheep** | 同形 |
| one pair of scissors | two pairs of scissors | 天生复数 |

## 常见陷阱

1. **把复数字尾一律读成 /s/**：dogs 是 /dɒɡz/ 不是 /dɒɡs/，days 是 /deɪz/ 不是 /deɪs/。中文没有 /s/ vs /z/ 对立，必须有意识地练。

2. **sheep 加 -s**：sheep 永远是 sheep，永远不加 -s。

3. **trousers 当单数用**：必须说 The trousers **are** too long. 不能说 The trousers **is**。
