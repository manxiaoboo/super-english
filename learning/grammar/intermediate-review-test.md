## 测试结果

7题中3题完全正确，4题有瑕疵。薄弱点集中在三类：

### 薄弱点1：情态动词的双重身份（推测 vs 义务）
- must have done → 只会是推测（"肯定做过"），不是义务
- 所有 "情态动词 + have + 过去分词" 结构都是推测用法

### 薄弱点2：时态的精确区分（完成 vs 完成进行）
- 完成时（have done）：强调结果/完成
- 完成进行时（have been doing）：强调过程/持续时长
- 有 been + -ing 就是进行标记

### 薄弱点3：虚拟语气的动词形式
- would 后永远跟动词原形（would travel, 不是 would gone）
- 对现在虚拟：If + 过去式, would + 原形
- 不规则动词过去式需要巩固（know→knew→known）

## 强项
