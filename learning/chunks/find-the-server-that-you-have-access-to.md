## Chinese Intention

查找你有权限访问的服务器

## Natural Chunk

find the server that you have access to

## Avoid Saying

- look up the server which you can visit

## Sentence Patterns

- You can find the ... server that you have access to.
- First, find the server that you have access to.

## Real-Life Examples

- You can find the Panzura server that you have access to.
- First, find the server that you have access to and then check the logs.

## Usage Notes

查找某台可访问的服务器时，find ... that you have access to 比 look up ... which you can visit 更自然。
