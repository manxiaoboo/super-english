# the old system is inefficient

- 学习日期：2026-07-08
- 熟练度：new
- 下次复习：2026-07-09
- 复习次数：0
- 最近结果：learned

## Chinese Intention

旧系统效率低

## Natural Chunk

the old system is inefficient

## Avoid Saying

- the old system has a low efficiency

## Sentence Patterns

- ... because the old system is inefficient.
- The old system is inefficient, so ...

## Real-Life Examples

- We should replace it because the old system is inefficient.

## Usage Notes

Say a system is inefficient. Has a low efficiency is understandable but stiff and uncommon.

## Production Prompts

1. 用 because the old system is inefficient 写一句说明更换系统原因的句子。
