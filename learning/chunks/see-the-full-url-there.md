## Natural Chunk

## Avoid Saying

## Sentence Patterns

- You can see the full URL there.

## Real-Life Examples

- This is the source file, and you can see the full URL there.
- Check the source page and you can see the full URL there.

## Usage Notes

说链接是否完整时，full URL 比 entire URL 更自然。
