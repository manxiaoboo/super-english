## Chinese Intention

## Natural Chunk

start your adaptation work in advance

## Avoid Saying

- to doing adoption work in advance

## Sentence Patterns

- You can refer to [sth] and start your adaptation work in advance.
- We're sharing this early so you can start your adaptation work ahead of time.

## Real-Life Examples

- We've shared the package.json so you can refer to it and start your adaptation work in advance.
- Here's the migration guide — feel free to start your adaptation work early.

## Usage Notes

adaptation work = adjusting/modifying code to be compatible; adoption = deciding to use something new. After 'to' (infinitive), use base form, not -ing.
