# see the real URL in it

- 下次复习：2026-07-09

## Chinese Intention

在文档里看到真实链接

## Natural Chunk

see the real URL in it

## Avoid Saying

- see the thuthy URL right here

## Sentence Patterns

- You can see the real URL in it.
- You can see the real URL right in the document.

## Real-Life Examples

- This is the original Markdown document, and you can see the real URL in it.
- Open the source file and you can see the real URL right in the document.

## Usage Notes

real URL 很自然；truthy 是编程语境词，不用来表示“真实的链接”。

## Production Prompts

1. 用 You can see the real URL in it. 造一句和网页、文档或源码有关的英文句子。
